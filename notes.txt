- create a sh file that loops through all years grab_287gfactsheets.sh
- make it executable chmod +x grab_287gfactsheets.sh
- run it and store log file to keep track of skipped files ./grab_287gfactsheets.sh > run_grab_287gfactsheets.log 2>&1
